# FastFurnace

A Minecraft plugin that makes furnaces smelt items faster.

## Building the project

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/FastFurnace.git
   cd FastFurnace
   ```
2. Build the plugin using Maven:
   ```bash
   mvn clean package
   ```
3. The compiled `.jar` file will be located in the `target/` directory.
